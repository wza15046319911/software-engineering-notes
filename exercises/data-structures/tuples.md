## 示例练习

### 练习1：创建和访问元组

**题目描述**：创建一个包含五个城市名称的元组，并演示如何访问其中的元素。

```python
# 创建一个城市元组
cities = ("北京", "上海", "广州", "深圳", "杭州")

# 访问元组元素
print(f"第一个城市: {cities[0]}")
print(f"最后一个城市: {cities[-1]}")
print(f"前三个城市: {cities[:3]}")
```
