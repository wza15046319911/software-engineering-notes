```python
def sum_even(numbers):
    """
    计算列表中所有偶数的和
    
    参数:
    numbers (list): 整数列表
    
    返回:
    int: 所有偶数的和
    """
    total = 0
    for num in numbers:
        if num % 2 == 0:
            total += num
    return total
```
